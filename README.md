# TYPING TEST 

🌐 A browser-based typing speed test built with vanilla HTML, CSS and Javascript

## Live Demo 
Check it out here: [typing test ⌨️](https://Ishani-Nair.github.io/typing-test/)

## Screenshot
![Typing Test Screenshot](screenshot.png)

## Features

- ⌛ Timed tests (30s, 1 min, 2 min, 5 min)
- 📉 Adjusted WPM — penalizes errors for a more accurate score
- 🙈 Blind mode — hides your input to build real muscle memory
- 🔴 Space error detection — highlights incorrect spaces visually
- 🏆 High score tracker — saves your best WPM locally
- 🔆 Light/dark mode toggle
- 📊 50+ random sentences to keep tests fresh

## Built With 
HTML, CSS and JavaScript - no frameworks or libraries

## How to Run 
Clone the repo and open `index.html` in your browser:
```bash
git clone https://github.com/GiornoGiovanna12345/typing-test.git
cd typing-test
open index.html
```
